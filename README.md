## Description

XRCloud Hubs API Documents

## Local Development

```console
$ bun i
$ bun start

// if you want to start your language, for example korean:
$ bun start --locale ko
```

and open http://localhost:3000

## How to add language?

- copy `i18n/ko` and paste your language
- Fix messages in your language directory
