## Scenes 리스트 조회

`https://hubs-api.dev.xrcloud.belivvr.com/v1/scenes?take=`<Bold>$\{take\}</Bold>`&page=`<Bold>$\{page\}</Bold>

:::info 요청 `URL 쿼리 파라미터`

- `take`: 한 페이지에 보여줄 Room 개수
- `page`: 페이지 번호

:::

:::info 응답

- Status: `200 OK`
- Content-Type: `application/json`

```json
{
  "count": 10,
  "data": [
    {
      "id": "...", // uuid
      "name": "...",
      "thumbnailUrl": "https://...",
      "modelUrl": "https://...", // nullable
      "projectUrl": "https://...",
      "accessKey": "...", // uuid
      "created": "...",
      "updated": "..."
    },
    ...
  ]
}
```

:::
