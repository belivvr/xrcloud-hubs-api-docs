## Scenes 리스트 조회

> **[GET]** `/v1/scenes?take=`<Bold>$\{take\}</Bold>`&page=`<Bold>$\{page\}</Bold>

:::info 요청

#### Headers
- Authorization: Bearer {accessToken}

#### URL 쿼리 파라미터
- `take`: 한 페이지에 보여줄 데이터 수
- `page`: 페이지 번호
- `custom-data`: 필터링 할 커스텀 데이터
- `custom-data-disabled`: `custom-data`를 사용하지 않고 모든 데이터를 조회

:::

:::info 응답

- Status: `200 OK`
- Content-Type: `application/json`

```json
{
  "count": 10,
  "data": [
    {
      "id": "...", // uuid
      "name": "...",
      "thumbnailUrl": "https://...",
      "modelUrl": "https://...", // nullable
      "projectUrl": "https://...",
      "custom-data": "...",
      "accessKey": "...", // uuid
      "created": "...",
      "updated": "..."
    },
    ...
  ]
}
```

:::
