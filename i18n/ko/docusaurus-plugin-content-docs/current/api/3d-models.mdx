## 3D Model 리스트 조회

> **[GET]** `/v1/3d-models?take=`<Bold>$\{take\}</Bold>`&page=`<Bold>$\{page\}</Bold>

:::info 요청 `URL Query Parameters`

- `take`: number of rooms to return
- `page`: page number to return

:::

:::info 응답

- Status: `200 OK`
- Content-Type: `application/json`

```json
{
  "count": 10,
  "data": [
    {
      "id": "3d-model uuid",
      "name": "3d-model name",
      "thumbnailUrl": "https://...",
      "glbUrl": "https://...",
      "active": false, // boolean
      "accessKey": "project uuid",
      "created": "created date",
      "updated": "latest update date"
    },
    ...
  ]
}
```

:::

## 3D Model 생성

> **[POST]** `/v1/3d-models`

:::info 요청

> Content-Type: `multipart/form-data; boundary=----WebKitFormBoundary...`

- glb: base64 Buffer (예: data:application/octet-stream;base64,...)
- thumbnail: base64 Buffer (예: data:application/octet-stream;base64,...)
- name: string
- active: boolean

:::

:::info 응답

- Status: `201 Created`
- Content-Type: `application/json`

```json
{
  "id": "3d-model uuid",
  "name": "3d-model name",
  "thumbnailUrl": "https://...",
  "glbUrl": "https://...",
  "active": false, // boolean
  "accessKey": "project uuid",
  "created": "created date",
  "updated": "latest update date"
}
```

:::
