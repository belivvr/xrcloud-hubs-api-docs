## Permission 리스트 조회

> **[GET]** `/v1/permissions?take=`<Bold>$\{take\}</Bold>`&page=`<Bold>$\{page\}</Bold>

:::info 요청

#### Headers
- Authorization: Bearer {accessToken}

#### URL 쿼리 파라미터
- `take`: 한 페이지에 보여줄 데이터 수
- `page`: 페이지 번호
- `custom-data`: 필터링 할 커스텀 데이터
- `custom-data-disabled`: `custom-data`를 사용하지 않고 모든 데이터를 조회

:::

:::info 응답

- Status: `200 OK`
- Content-Type: `application/json`

```json
{
  "count": 10,
  "data": [
    {
      "id": "permission uuid", // uuid
      "name": "permission name",
      "pinObject": false, // boolean. 물체 고정 권한
      "fly": false, // boolean. 비행 모드 권한
      "kickUsers": false, // boolean. 유저 추방 권한
      "muteUsers": false, // boolean. 유저 음소거 권한
      "amplifyAudio": false, // boolean. 오디오 강화 권한
      "spawnAndMoveMedia": false, // boolean. 미디어 생성 및 이동 권한
      "spawnDrawing": false, // boolean. 펜 생성 및 이용 권한
      "spawnCamera": false, // boolean. 카메라 생성 및 이용 권한
      "spawnEmoji": false, // boolean. 이모지 사용 권한
      "accessKey": "프로젝트 uuid",
      "created": "생성 날짜",
      "updated": "마지막 업데이트 날짜"
    },
    ...
  ]
}
```

:::
