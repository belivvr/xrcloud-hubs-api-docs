## Get many Permissions

> **[GET]** `/v1/permissions?take=`<Bold>$\{take\}</Bold>`&page=`<Bold>$\{page\}</Bold>

:::info Request

#### Headers
- Authorization: Bearer {accessToken}

#### URL Query Parameters
- `take`: number of rooms to return
- `page`: page number to return
- `custom-data`: custom data to filter
- `custom-data-disabled`: disable `custom-data` and return all data

:::

:::info Response

- Status: `200 OK`
- Content-Type: `application/json`

```json
{
  "count": 10,
  "data": [
    {
      "id": "permission uuid", // uuid
      "name": "permission name",
      "pinObject": false, // boolean
      "fly": false, // boolean
      "kickUsers": false, // boolean
      "muteUsers": false, // boolean
      "amplifyAudio": false, // boolean
      "spawnAndMoveMedia": false, // boolean
      "spawnDrawing": false, // boolean
      "spawnCamera": false, // boolean
      "spawnEmoji": false, // boolean
      "custom-data": "custom data",
      "accessKey": "project uuid",
      "created": "created date",
      "updated": "latest update date"
    },
    ...
  ]
}
```

:::

- Content-Type: application/json
