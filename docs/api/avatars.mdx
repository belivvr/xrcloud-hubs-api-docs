## Get many Avatars

> **[GET]** `/v1/avatars?take=`<Bold>$\{take\}</Bold>`&page=`<Bold>$\{page\}</Bold>

:::info Request

#### Headers
- Authorization: Bearer {accessToken}

#### URL Query Parameters
- `take`: number of rooms to return
- `page`: page number to return
- `custom-data`: custom data to filter
- `custom-data-disabled`: disable `custom-data` and return all data

:::

:::info Response

- Status: `200 OK`
- Content-Type: `application/json`

```json
{
  "count": 10,
  "data": [
    {
      "id": "avatar uuid", // uuid
      "name": "avatar name",
      "thumbnailUrl": "https://...",
      "glbUrl": "https://...",
      "active": false, // boolean
      "customData": "custom data",
      "accessKey": "project uuid",
      "created": "created date",
      "updated": "latest update date"
    },
    ...
  ]
}
```

:::

## Create a Avatar

> **[POST]** `/v1/avatars`

:::info Request

#### Headers
- Authorization: Bearer {accessToken}
- Content-Type: `multipart/form-data; boundary=----WebKitFormBoundary...`

#### Body
- glb: base64 Buffer (example: data:application/octet-stream;base64,...)
- thumbnail: base64 Buffer (example: data:application/octet-stream;base64,...)
- name: string
- active: boolean
- custom-data: string

:::

:::info Response

- Status: `201 Created`
- Content-Type: `application/json`

```json
{
  "id": "avatar uuid",
  "name": "avatar name",
  "thumbnailUrl": "https://...",
  "glbUrl": "https://...",
  "active": false, // boolean
  "customData": "custom data",
  "accessKey": "project uuid",
  "created": "created date",
  "updated": "latest update date"
}
```

:::
