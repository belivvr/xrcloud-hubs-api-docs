## Get many Avatars

`/v1/avatars?take=`<Bold>$\{take\}</Bold>`&page=`<Bold>$\{page\}</Bold>

:::info Request `URL Query Parameters`

- `take`: number of rooms to return
- `page`: page number to return

:::

:::info Response

- Status: `200 OK`
- Content-Type: `application/json`

```json
{
  "count": 10,
  "data": [
    {
      "id": "avatar uuid", // uuid
      "name": "avatar name",
      "thumbnailUrl": "https://...",
      "glbUrl": "https://...",
      "active": false, // boolean
      "accessKey": "project uuid",
      "created": "created date",
      "updated": "latest update date"
    },
    ...
  ]
}
```

:::
