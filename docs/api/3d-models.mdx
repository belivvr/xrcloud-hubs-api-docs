## Get many 3D Models

> **[GET]** `/v1/3d-models?take=`<Bold>$\{take\}</Bold>`&page=`<Bold>$\{page\}</Bold>

:::info Request `URL Query Parameters`

- `take`: number of rooms to return
- `page`: page number to return

:::

:::info Response

- Status: `200 OK`
- Content-Type: `application/json`

```json
{
  "count": 10,
  "data": [
    {
      "id": "3d-model uuid",
      "name": "3d-model name",
      "thumbnailUrl": "https://...",
      "glbUrl": "https://...",
      "active": false, // boolean
      "accessKey": "project uuid",
      "created": "created date",
      "updated": "latest update date"
    },
    ...
  ]
}
```

:::

## Create a 3D Model

> **[POST]** `/v1/3d-models`

:::info Request

> Content-Type: `multipart/form-data; boundary=----WebKitFormBoundary...`

- glb: base64 Buffer (example: data:application/octet-stream;base64,...)
- thumbnail: base64 Buffer (example: data:application/octet-stream;base64,...)
- name: string
- active: boolean

:::

:::info Response

- Status: `201 Created`
- Content-Type: `application/json`

```json
{
  "id": "3d-model uuid",
  "name": "3d-model name",
  "thumbnailUrl": "https://...",
  "glbUrl": "https://...",
  "active": false, // boolean
  "accessKey": "project uuid",
  "created": "created date",
  "updated": "latest update date"
}
```

:::
