## Get many Scenes

> **[GET]** `/v1/scenes?take=`<Bold>$\{take\}</Bold>`&page=`<Bold>$\{page\}</Bold>

:::info Request

#### Headers
- Authorization: Bearer {accessToken}

#### URL Query Parameters
- `take`: number of rooms to return
- `page`: page number to return

:::

:::info Response

- Status: `200 OK`
- Content-Type: `application/json`

```json
{
  "count": 10,
  "data": [
    {
      "id": "...", // uuid
      "name": "...",
      "thumbnailUrl": "https://...",
      "modelUrl": "https://...", // nullable
      "projectUrl": "https://...",
      "accessKey": "...", // uuid
      "created": "...",
      "updated": "..."
    },
    ...
  ]
}
```

:::
